【こんな方におすすめ】
 - 配信画面をおしゃれにしたい方
 - デザインの知識がなくても使いたい方
 - OBSで手軽にシーンを切り替えたい方
 - VTuber・ゲーム実況・雑談配信をしている方

【主な特徴】

  ■ 4つのシーンタイプ
  配信準備中 / もうすぐ開始 / 離席中 / 配信終了
  ワンクリックでシーンを切り替え！

  ■ 5つのプロ品質テーマ
  ネオン / ゲーミング / ミニマル / パステル / レトロ
  テーマを選ぶだけで一瞬で雰囲気が変わります

  ■ 豊富なカスタマイズ
  ・10種類のフォント
  ・グロー / グラデーション / 縁取り / シャドウ
  ・6種類のアニメーション
  ・4種類のパーティクル（丸・星・雪・バブル）
  ・カウントダウンタイマー付き
  ・プログレスバー対応
  ・枠線（グロー効果付き）

  ■ かんたん操作
  ・設定画面でリアルタイムプレビュー
  ・URLをコピーしてOBSに貼るだけ
  ・プリセット5つまで保存可能
  ・設定のエクスポート/インポート対応

  ■ 完全オフライン対応
  ・HTMLファイルをブラウザで開くだけ
  ・インストール不要
  ・YouTube / Twitch / ニコ生 全対応

【動作環境】
  ・OBS Studio（推奨）
  ・Google Chrome / Edge / Firefox
  ・Windows / Mac / Linux対応

【使い方】
  1. settings.html をブラウザで開く
  2. テーマ・テキスト・エフェクトを設定
  3. OBS用URLをコピー
  4. OBSのブラウザソースにURLを貼り付け
  → 完成！
