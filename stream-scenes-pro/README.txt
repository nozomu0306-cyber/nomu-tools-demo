==========================================
  Stream Scenes Pro - 配信画面セットビルダー
  v1.0.0
==========================================

■ 概要
OBSのブラウザソースで使える、配信用シーン画面の
ビルダーツールです。
待機画面・開始画面・休憩画面・終了画面の4種類を
美しいエフェクトとアニメーションで演出できます。


■ 使い方

【STEP 1】設定を行う
  1. 「settings.html」をブラウザ（Chrome推奨）で開く
  2. シーンの種類・テーマ・テキスト・エフェクトを設定
  3. プレビューでリアルタイムに確認できます

【STEP 2】OBSに追加する
  1. 設定画面上部の「OBS用URL」をコピー
  2. OBSの「ソース」→「＋」→「ブラウザ」
  3. 「ローカルファイル」のチェックを【外す】
  4. URLを貼り付ける
  5. 幅・高さを設定（推奨: 1920×1080 または 1280×720）

【STEP 3】シーンを切り替える
  - 各シーン（待機・開始・休憩・終了）ごとに
    別々のURLを生成し、OBSのシーンに配置できます
  - 設定画面でシーンを切り替え → URLコピー → OBSに設定


■ 搭載機能

  【シーン】
  ・配信準備中 (Waiting)
  ・もうすぐ開始 (Starting Soon)
  ・離席中 (BRB - Be Right Back)
  ・配信終了 (Ending)

  【テーマ】5種類
  ・ネオン ....... 青い光彩のSF/サイバー風
  ・ゲーミング ... 虹色グラデーション
  ・ミニマル ..... シンプル・モダン
  ・パステル ..... やわらかい暖色系
  ・レトロ ....... 緑のドットフォント風

  【テキスト】
  ・メインテキスト / 上下ラベル
  ・10種類のフォント
  ・サイズ・色の個別指定

  【エフェクト】
  ・グロー（光彩）.... 色・サイズ指定
  ・グラデーション文字 .... 2色指定
  ・アウトライン（縁取り）.... 色・太さ指定
  ・ドロップシャドウ

  【アニメーション】
  ・フェードイン（上/下）
  ・パルス / グロー / フロート
  ・シマー（グラデーション連動）

  【背景】
  ・アニメーショングラデーション
  ・静的グラデーション / 単色
  ・透過（OBSクロマキー不要）
  ・パーティクル（丸/星/雪/バブル）

  【カウントダウン】
  ・開始までのカウントダウンタイマー
  ・MM:SS / HH:MM:SS / 秒のみ
  ・サイズ・色の個別指定

  【プログレスバー】
  ・カウントダウン連動の進捗バー
  ・色・高さのカスタマイズ

  【枠線】
  ・色・幅・角丸・グロー効果

  【保存・管理】
  ・プリセット5つまで保存
  ・JSON エクスポート/インポート


■ ファイル構成

  stream-scenes-pro/
  ├── index.html ......... シーン表示（OBSに指定）
  ├── settings.html ...... 設定画面（ブラウザで開く）
  ├── css/
  │   ├── scene.css ...... シーンのスタイル
  │   └── settings.css ... 設定画面のスタイル
  ├── js/
  │   ├── scene.js ....... シーン描画ロジック
  │   └── settings.js .... 設定ロジック
  └── README.txt ......... このファイル


■ 注意事項

  ・Google Chrome / OBS Studio でのご利用を推奨
  ・OBSでは「ローカルファイル」ではなくURL方式を使用
  ・フォントの読み込みにインターネット接続が必要
  ・設定はブラウザのローカルストレージに保存


■ トラブルシューティング

  Q: シーンが表示されない
  A: OBSのブラウザソースで「ローカルファイル」の
     チェックが外れているか確認してください。

  Q: テーマを変更したのに反映されない
  A: 設定画面で変更後「URLをコピー」を押し、
     OBSに新しいURLを貼り直してください。

  Q: 設定を別PCに移したい
  A: 「保存」タブの「エクスポート」でJSONを保存し、
     別PCで「インポート」してください。

  Q: カウントダウンが動かない
  A: 「詳細」タブで「カウントダウンを表示」に
     チェックが入っているか確認してください。
