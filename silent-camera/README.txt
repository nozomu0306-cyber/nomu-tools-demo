Silent Camera — 無音カメラ
=========================

iPhone のシャッター音を鳴らさずに写真・ビデオを撮影できる
ブラウザアプリです。寝ている子どもやペットの撮影に。

■ 使い方
  1. HTTPS でホストされた index.html を Safari で開く
     (GitHub Pages / localhost でもOK)
  2. 初回、カメラ（必要ならマイク）の使用を許可する
  3. 丸い白いボタンで撮影／赤で録画開始・停止
  4. 左下のサムネから保存または共有

■ なぜ音が鳴らないか
  iPhone のシャッター音強制は、純正カメラアプリ (AVFoundation の
  静止画キャプチャ経路) にかかる仕様です。
  Safari の getUserMedia + Canvas / MediaRecorder は別経路のため、
  OS のシャッター音は鳴りません。

■ 機能
  - 写真 / ビデオ モード切替
  - 前 / 後 カメラ切替
  - グリッド表示
  - フラッシュライト (対応端末のみ)
  - ズーム (対応端末のみ)
  - NIGHT モード (赤フィルタ / 暗闇でも目を慣らしたまま)
  - DIM スライダ (画面を暗くして周囲を明るくしない)
  - マイク ON / OFF (録画時のみ影響)
  - 共有シート / ダウンロード保存

■ 注意
  - HTTPS または localhost 必須 (getUserMedia 制約)
  - iOS Safari は MediaRecorder が新しい OS でのみ対応
  - ブラウザの動作仕様は OS / バージョンで変わるため、
    端末によっては一部機能が使えない場合があります
