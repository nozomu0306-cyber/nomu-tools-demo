==========================================
  Stream Soundboard - 配信用サウンドボード
  v1.0.0
==========================================

■ 概要
配信中にワンクリックで効果音を鳴らせる
サウンドボードアプリです。
キーボードショートカット対応で、
ゲーム中でも即座に音を鳴らせます。

■ 他のサウンドボードとの違い
  ・インストール不要！ブラウザで完結
  ・音声データはブラウザ内に安全に保存
  ・グループ分け + 検索で大量の音を管理
  ・20色のカラフルなパッドで視認性抜群
  ・エクスポート/インポートで設定の移行が簡単
  ・軽量で配信PCの負荷が少ない

■ 使い方

  1.「index.html」をブラウザ（Chrome推奨）で開く
  2.「＋ サウンド追加」で音声ファイルを登録
  3. パッドをクリックして再生！
  4. キーボードショートカットも設定可能

■ 機能一覧

  【パッド】
  ・クリックで即再生
  ・20色から選べるカラーパッド
  ・キーボードショートカット割当
  ・再生中はビジュアルフィードバック
  ・右クリックメニュー（編集/複製/削除）

  【グループ】
  ・カテゴリ分けで整理
  ・グループの追加/名前変更/削除
  ・ドラッグで並び替え

  【マスターボリューム】
  ・全体の音量を一括調整
  ・再生中でもリアルタイム反映

  【検索】
  ・名前で即検索・絞り込み

  【保存】
  ・全データのエクスポート（音声含む）
  ・インポートで完全復元
  ・ブラウザを閉じてもデータ保持

  【ショートカット】
  ・各パッドにキー割当可能
  ・ESCキーで全停止

■ 対応形式
  MP3 / WAV / OGG / FLAC / AAC / WebM

■ ファイル構成
  stream-soundboard/
  ├── index.html ......... メインアプリ
  ├── css/
  │   └── style.css ...... スタイル
  ├── js/
  │   └── app.js ......... アプリケーションロジック
  └── README.txt ......... このファイル

■ 注意事項
  ・Google Chrome 推奨
  ・音声データはIndexedDBに保存されます
  ・ブラウザのキャッシュクリアで消えます
  ・大事なデータはエクスポートしてください
