==========================================
  Stream Info Panel - 配信情報パネルビルダー
  v1.0.0
==========================================

■ 概要
OBSのブラウザソースで使える、配信情報表示用の
ウィジェットビルダーです。
7種類のウィジェットを自由に配置して、
配信画面に情報パネルを追加できます。

■ 他にない特徴
  ・複数ウィジェットを1つのブラウザソースに配置
  ・ウィジェットのXY座標を自由に指定
  ・テーマ連動で統一感のあるデザイン
  ・カウンターはpostMessage APIで外部連携可能

■ ウィジェット一覧

  📊 目標バー .... フォロワー/登録者の目標進捗
  ⏱  配信時間 .... 配信開始からの経過時間
  🔢 カウンター .. デスカウント等の数値表示
  📝 テキスト .... 自由テキスト表示
  📜 ティッカー .. スクロールテキスト
  🎉 イベント .... 最新フォロー/サブスク表示
  🔗 SNSリンク ... X/YouTube等のアカウント表示

■ 使い方

  1. settings.html をブラウザで開く
  2. テーマとフォントを選択
  3. ウィジェットを追加し、位置・内容を設定
  4. OBS用URLをコピーしてOBSに貼り付け

■ カウンターAPI

  外部スクリプトからpostMessageでカウンター操作可能：

  // インクリメント
  frame.contentWindow.postMessage(
    JSON.stringify({type:'counter-update', id:'counter1', action:'increment'}), '*'
  );
  // リセット
  frame.contentWindow.postMessage(
    JSON.stringify({type:'counter-update', id:'counter1', action:'reset'}), '*'
  );
