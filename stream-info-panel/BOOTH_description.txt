【ここが違う！唯一の複数ウィジェット対応】

  他のツールは「タイマーだけ」「カウンターだけ」。
  本ツールは1つのブラウザソースに
  複数のウィジェットを自由配置できます。

  OBSのソース数を減らしながら
  リッチな情報パネルを実現！

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【7種類のウィジェット】

  📊 目標バー ... フォロワー・登録者目標の進捗表示
  ⏱  配信時間 ... 配信開始からの経過時間
  🔢 カウンター . デスカウント・クリア回数等
  📝 テキスト ... お知らせ・スケジュール等
  📜 ティッカー . スクロールする告知テキスト
  🎉 イベント ... 最新フォロー・サブスク表示
  🔗 SNSリンク . X / YouTube等のアカウント

【5つのテーマ】
  ネオン / ゲーミング / ミニマル / パステル / レトロ
  ウィジェット全体に統一テーマを適用！

【高度な機能】
  ・XY座標で自由配置
  ・各ウィジェットのサイズ調整
  ・出現アニメーション（フェード・スライド）
  ・カウンターは外部APIで操作可能
  ・設定のエクスポート/インポート対応

【使い方】
  1. settings.html をブラウザで開く
  2. テーマ選択 → ウィジェットを追加
  3. 位置・内容をカスタマイズ
  4. OBS用URLをコピー → OBSに貼り付け

【動作環境】
  ・OBS Studio（推奨）
  ・Google Chrome / Edge / Firefox
  ・Windows / Mac / Linux 対応
