  Stream Label Pro - ラベル/ローワーサードビルダー
  バージョン 1.0
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

OBS Studio のブラウザソースで使える、
ラベル/ローワーサード/テロップオーバーレイツールです。

配信者名、SNS、スケジュール等を
おしゃれなデザインで配信画面に常時表示できます。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 特徴
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ★ 6種レイアウト
    ローワーサード / 名刺 / SNSバー /
    テロップ / ミニラベル / スケジュール

  ★ マルチスロット（最大3つ同時表示）
    名前ラベル + SNSバー + スケジュールを
    同時に画面上に配置可能

  ★ 5テーマ + 豊富なカスタマイズ
    ネオン / ゲーミング / ミニマル / パステル / レトロ

  ★ 入場/退場アニメーション
    スライド / フェード / 自動非表示タイマー

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 使い方（3ステップ）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. settings.html をブラウザで開く
  2. レイアウトを選んでテキスト・デザインを設定
  3. OBS用URLをコピー → ブラウザソースに貼り付け

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 外部API
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  window.StreamLabelPro.show(0)   // スロット0を表示
  window.StreamLabelPro.hide(0)   // スロット0を非表示
  window.StreamLabelPro.update(0, { line1: '新テキスト' })
