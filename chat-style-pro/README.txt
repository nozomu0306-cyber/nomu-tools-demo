==========================================
  Chat Style Pro - チャットオーバーレイ
  カスタマイザー  v1.0.0
==========================================

■ 概要
OBSのブラウザソースで使える、チャット表示用
オーバーレイのカスタマイズツールです。
7種類のテーマと豊富なカスタマイズで、
配信画面にマッチするチャット表示を作れます。


■ 使い方

【STEP 1】設定を行う
  1. 「settings.html」をブラウザ（Chrome推奨）で開く
  2. テーマ・フォント・表示オプションを設定
  3. デモモードでリアルタイムプレビュー

【STEP 2】OBSに追加する
  1. 設定画面上部の「OBS用URL」をコピー
  2. OBSの「ソース」→「＋」→「ブラウザ」
  3. 「ローカルファイル」のチェックを【外す】
  4. URLを貼り付ける
  5. 幅400〜500、高さ600〜800 程度を推奨

【STEP 3】実際のチャットと連携
  本ツールは以下の方法でチャットを表示できます：
  ・デモモード（設定確認用）
  ・postMessage API（外部連携用）
  ・YouTube Live CSS / Chat v2.0 と併用


■ テーマ一覧

  ・ネオン ....... 青い光彩のサイバー風
  ・ゲーミング ... ピンク系グラデーション
  ・ミニマル ..... シンプルモノトーン
  ・パステル ..... やわらかい暖色系
  ・レトロ ....... 緑のドットフォント風
  ・バブル ....... カラフルなグラデ背景
  ・ガラス ....... ぼかし付き半透明


■ カスタマイズ項目

  【テキスト】フォント4種、サイズ調整
  【表示】最大数 / 自動消去 / 間隔 / 角丸
  【カラー】名前色 / テキスト色 / 背景色 / 透明度
  【アニメ】フェード / スライド / スケール / バウンス
  【保存】プリセット3つ / JSON出力・入力
