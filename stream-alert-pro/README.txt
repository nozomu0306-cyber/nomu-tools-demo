  Stream Alert Pro - 配信アラートビルダー
  バージョン 1.0
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

OBS Studio のブラウザソースで使える、
高機能アラートオーバーレイツールです。

フォロー・サブスク・投げ銭・レイドなどの
アラートをプロ品質のアニメーションで表示できます。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 他にない特徴
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ★ ビジュアルタイムラインエディタ
    ドラッグ操作でアニメーションのタイミングを
    視覚的に編集。コーディング不要。

  ★ マルチレイヤー合成
    テキスト・絵文字・シェイプ・パーティクルを
    自由に重ねて1つのアラートを構成。

  ★ パーティクルエフェクト
    紙吹雪・花火・ハート・星・絵文字の雨など
    派手なエフェクトで配信を盛り上げます。

  ★ サウンド統合
    タイムライン上の任意のタイミングで
    効果音を再生できます。

  ★ 10種類以上のテンプレート
    ゲーミング・かわいい・ミニマル・VTuber等
    ワンクリックで即使えるプリセット付き。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 使い方（4ステップ）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. settings.html をブラウザ（Chrome推奨）で開く
  2. テーマを選択 → レイヤーを追加・編集
  3. タイムラインでアニメーションのタイミングを調整
  4. OBS用URLをコピー → OBSのブラウザソースに貼り付け

  ※ インストール不要！ブラウザで開くだけで使えます。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 設定画面の使い方
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  【アラートタブ】
  ・6種類のアラートタイプを選択
    （Follow / Subscribe / Donation / Raid / Custom1 / Custom2）
  ・タイプごとに有効/無効を切り替え
  ・表示時間と優先度を設定

  【レイヤータブ】
  ・テキスト / 絵文字 / シェイプ / パーティクルの
    レイヤーを追加・編集・削除
  ・レイヤーごとに位置・サイズ・色・エフェクトを設定
  ・テキストには変数を使用可能:
    {username} … ユーザー名
    {amount}   … 金額
    {message}  … メッセージ
    {tier}     … サブスクティア
    {viewers}  … レイド人数

  【タイムラインタブ】
  ・各レイヤーの表示タイミングをドラッグで調整
  ・再生ボタンでプレビュー確認
  ・総再生時間の変更

  【テーマタブ】
  ・5種類のテーマ: ネオン / ゲーミング / ミニマル / パステル / レトロ
  ・フォント選択（日本語対応フォント含む）
  ・アラート表示位置の変更（9箇所）

  【サウンドタブ】
  ・アラートタイプごとに効果音をアップロード
  ・対応形式: MP3 / WAV / OGG / FLAC / AAC
  ・音量・フェードイン/アウト設定

  【キュータブ】
  ・アラート間の間隔（0〜10秒）
  ・最大キューサイズ
  ・優先アラートのキュースキップ

  【保存タブ】
  ・プリセット5枠まで保存
  ・設定のエクスポート/インポート（JSON形式）
  ・テンプレートギャラリーからワンクリック適用

■ OBS設定方法
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. OBS Studioを開く
  2. ソースの「＋」→「ブラウザ」を追加
  3. 「ローカルファイル」をOFFにする
  4. URLに設定画面でコピーしたURLを貼り付け
  5. 幅: 1920 / 高さ: 1080（推奨）
  6. 「OK」で完了

  ※ 透明背景に対応しているため、
    他のソースの上に重ねて使えます。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 外部連携API（上級者向け）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  JavaScript API:
  ─────────────
  // ブラウザコンソールまたは外部スクリプトから:
  window.StreamAlertPro.trigger('follow', {
    username: 'ユーザー名'
  });

  window.StreamAlertPro.trigger('donation', {
    username: 'ユーザー名',
    amount: '500',
    message: 'がんばって！'
  });

  window.StreamAlertPro.trigger('subscribe', {
    username: 'ユーザー名',
    tier: 'Tier 1',
    message: 'いつも楽しみにしてます'
  });

  window.StreamAlertPro.trigger('raid', {
    username: 'ユーザー名',
    viewers: '42'
  });

  // アラートをクリア
  window.StreamAlertPro.clear();


  postMessage API:
  ────────────────
  // iframe内のアラートに外部からメッセージを送信:
  iframe.contentWindow.postMessage({
    type: 'trigger',
    alertType: 'follow',
    username: 'ユーザー名'
  }, '*');
